🚀 Google表單自動填寫擴充套件

輕鬆一鍵填寫表單，節省時間，提升效率！

🔥 點此前往 Chrome 商城安裝（完全免費）

🌟 為什麼你需要這個擴充功能？

這是一個能讓你「一鍵自動填好表單」的神奇工具！
不需要懂複雜的電腦設定，只要提早輸入好你要填寫的內容，時間一到或打開表單時，系統就會瞬間幫你把所有格子填滿，讓你永遠比別人快一步！

超適合以下情境：

  • 搶報名、搶票、限量周邊： 超快速一次填寫完設定的內容，自動送出！
  • 藝人/模特兒： 申請試鏡、通告。
  • 業務： 客戶資料填報、拜訪表。
  • 自由工作者： 合作提案報名。
  • 學生： 課程報名、活動表單。

📖 如何使用 (安裝步驟)

  1. 點擊上方的 Chrome 商城連結，進入擴充功能安裝頁面。
  2. 點擊「加到 Chrome」將它安裝到您的瀏覽器。
  3. 開啟你要填寫的 Google 表單。
  4. 點擊瀏覽器右上角的擴充功能圖示，把你想預先準備好的資料（關鍵字與答案）填入並存檔。
  5. 下次再次打開同一個表單時，資料就會自動填寫完畢！

💡 詳細設定範本與精選範例

擴充功能會使用「關鍵字」來尋找題目，只要表單題目字眼包含您設定的關鍵字，系統就會自動將對應的「填寫內容」填入或勾選。

1️⃣ 電子郵件 (Email)
  • 設定「關鍵字」：mail
  • 設定「填寫內容」：user@gmail.com
  • 🎯 實際效果：不論表單上的題目是「Email」或「電子郵件」，只要包含 mail（不分大小寫），就會自動幫您填入信箱。
2️⃣ 複選題的多個答案 (多重勾選 Checkboxes)
  • 設定「關鍵字」：語言
  • 設定「填寫內容」：中文,英文,日文 (請用英文半形逗號隔開多個選項)
  • 🎯 實際效果：當遇到多選題核取方塊時，會自動幫你同時勾選「中文」、「英文」及「日文」。
3️⃣ 單選題或下拉式選單 (Radio / Dropdown)
  • 設定「關鍵字」：性別
  • 設定「填寫內容」：男
  • 🎯 實際效果：只要該選項裡有標示「男」，系統就會自動點擊或選中該選項。
4️⃣ 一般短答輸入 (文字 Text / Name)
  • 設定「關鍵字」：姓名
  • 設定「填寫內容」：王小明
  • 🎯 實際效果：題目如「報名者姓名」、「您的真實姓名」，會自動填入王小明。
5️⃣ 電話號碼 (Phone number)
  • 設定「關鍵字」：電話
  • 設定「填寫內容」：0912345678
  • 🎯 實際效果：遇到「聯絡電話」、「手機號碼」等欄位，都會自動帶入號碼。

❓ 常見問題 (FAQ)

Q: 如何確認擴充功能正常運作？
A: 開啟 Google 表單時，文字欄位會自動填入你儲存在擴充功能內的答案。如果沒有，請檢查：
  • 擴充功能是否已在瀏覽器開啟
  • 設定面板中的關鍵字是否已正確儲存
  • 關鍵字是否對應表單上的題目文字
Q: 想更改自動填入的內容怎麼辦？
A: 自動填入只會幫你節省打字時間，不會強制鎖定！你隨時可以直接在表單欄位上手動修改，送出前請記得再次確認即可。
Q: 還需要下載 QA.txt 或執行 .bat 檔案嗎？
A: 絕對不用！此版本已進化為「線上正式版」，所有的關鍵字設定都會直接儲存在您的 Chrome 瀏覽器中，安全又快速，不需要再處理任何程式碼檔案。

🚀 Google Form Auto-Fill Extension

Fill forms effortlessly with just one click, save time, and be one step ahead!

🔥 Install for free from Chrome Web Store

🌟 Why do you need this?

A magical tool that fills out forms for you!
No technical skills required. Set your content in advance, and when you open a Google Form, everything will automatically fill instantly so you're always one step ahead.

Highly recommended for:

  • Snagging tickets & limited items: Auto-fill everything super fast!
  • Artists/Actors: Audition and casting applications.
  • Sales/Business: Quick data entry for customer details.
  • Freelancers: Proposals and applications.
  • Students: Registering for popular classes or events.

📖 How to use (Installation)

  1. Click the Chrome Web Store link above.
  2. Click "Add to Chrome" to install it.
  3. Open any Google Form you want to fill.
  4. Click the extension icon in the upper right corner to input your keywords and answers, then save.
  5. Next time you open the form, the data will be automatically filled!

💡 Configuration Examples

The extension uses "Keywords". If a form question contains your keyword, it automatically interacts with the corresponding field!

1️⃣ Email
  • Set Keyword: mail
  • Set Fill-in content: user@example.com
  • 🎯 Effect: Matches "Email", "Your e-mail", filling it instantly!
2️⃣ Checkboxes (Multiple Answers)
  • Set Keyword: Language
  • Set Fill-in content: English,Spanish,French (Use a comma to separate)
  • 🎯 Effect: Checks all three options matching these words simultaneously.

❓ FAQ

Q: Do I need to download QA.txt code?
A: No! This is the official version which runs completely in the browser. You configure everything straight from the Chrome extension popup seamlessly!
