安裝步驟 Installation Steps

  1. 下載並解壓縮檔案
    Download and extract files
  2. 開啟Chrome瀏覽器，進入擴充功能管理頁面
    Open Chrome browser, go to extension management
  3. 開啟開發人員模式
    Enable Developer mode
  4. 點選「載入未封裝項目」
    Click "Load unpacked"
  5. 選擇解壓縮後的資料夾
    Select the extracted folder

QA.txt 設定說明 Setting Guide

格式 Format:
題目，答案
Question text, Answer

範例 Example:

name,blues
姓名,王小明
電話,0912345678

當表單題目包含「姓名」時，會自動填入「王小明」
When the form question contains "姓名", it will auto-fill "王小明"

注意事項 Notes

  • 目前僅支援文字輸入框
    Currently only supports text input fields
  • 修改QA.txt後需重新執行更新批次檔
    After editing QA.txt, run the update batch file
  • 送出前請檢查填寫內容
    Please check the content before submission
