  1. 點擊上方的 Chrome 商城連結，進入擴充功能安裝頁面。

💡 詳細設定範本與精選範例

擴充功能會使用「關鍵字」來尋找題目，只要表單題目字眼包含您設定的關鍵字，系統就會自動將對應的「填寫內容」填入或勾選。

1️⃣ 電子郵件 (Email)
  • Key: mail
  • Value: user@gmail.com
  • 🎯 Effect: 不論表單上的題目是「Email」或「電子郵件」，只要包含 mail（不分大小寫），就會自動幫您填入信箱。
2️⃣ 複選題的多個答案 (多重勾選 Checkboxes)
  • Key: 語言

3️⃣ 單選題或下拉式選單 (Radio / Dropdown)
  • Key: 性別
  • Value: 男
  • 🎯 Effect: 只要該選項裡有標示「男」，系統就會自動點擊或選中該選項。
4️⃣ 一般短答輸入 (文字 Text / Name)
  • Key: 姓名
  • Value: 王小明

  • Key: 電話
  • Value: 0912345678
  • 🎯 Effect: 遇到「聯絡電話」、「手機號碼」等欄位，都會自動帶入號碼。

  • 設定面板中的關鍵字是否已正確儲存

🚀 Preenchimento Automático do Google Forms

Uma ferramenta mágica para preencher formulários para você em apenas um clique!

🔥 Instale gratuitamente na Chrome Web Store

🌟 Por que você precisa disso?

Uma ferramenta mágica para formulários!
Defina seu conteúdo com antecedência e, ao abrir um Google Forms, tudo será preenchido perfeitamente.

Altamente recomendado para:

  • Ferramenta de assistência para pessoas com deficiência: Reduz o esforço de digitação e supera a exclusão digital!
  • Disputar ingressos ou inscrições concorridas: Preencha tudo super rápido de uma vez e seja o primeiro!
  • Artistas: Inscrições para testes ou casting.
  • Vendedores: Lançamento rápido de dados.
  • Freelancers: Propostas de clientes.
  • Estudantes: Inscrições em eventos e disciplinas.

📖 Como Usar (Instalação)

  1. Clique no link da Chrome Web Store acima.
  2. Clique em 'Usar no Chrome'.
  3. Abra um formulário escolhido.
  4. Configure suas palavras-chave na extensão e salve.
  5. Tudo será Preenchido automaticamente a partir de agora!
