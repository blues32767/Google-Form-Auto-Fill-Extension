安裝步驟

Installation Steps

  1. 下載並解壓縮檔案 - 點擊「立即下載」按鈕獲取檔案
  2. Download and extract files - Click "Download Now" to get the file
  3. 開啟Chrome瀏覽器，進入擴充功能管理頁面 - 輸入 chrome://extensions/ 即可進入
  4. Open Chrome browser, go to extension management - Enter chrome://extensions/ to access
  5. 開啟開發人員模式 - 位於頁面右上角的開關
  6. Enable Developer mode - Switch located at the top-right corner
  7. 點選「載入未封裝項目」 - 選擇您解壓後的資料夾
  8. Click "Load unpacked" - Select your extracted folder
  9. 選擇解壓縮後的資料夾 - 確認後即可完成安裝
  10. Select the extracted folder - Confirm to complete installation

QA.txt 設定說明

Setting Guide

格式:
題目，答案
簡單易懂，每行一組配對 Format:
Question text, Answer
Simple and clear, one pair per line

範例:

Example:

name,blues
姓名,王小明
電話,0912345678

當表單題目包含「姓名」時，會自動填入「王小明」
支援部分關鍵字匹配，靈活應用 When the form question contains "姓名", it will auto-fill "王小明"
Supports partial keyword matching for flexible use

注意事項

Notes

  • 目前僅支援文字輸入框 - 未來可能新增更多功能
  • Currently only supports text input fields - More features may be added in the future
  • 修改QA.txt後需重新執行更新批次檔 - 確保答案即時更新
  • After editing QA.txt, run the update batch file - Ensure answers are updated instantly
  • 送出前請檢查填寫內容 - 以避免意外錯誤
  • Please check the content before submission - To avoid unintended errors

如需更新答案，請按照以下步驟操作：

To update answers, follow these steps:

  1. 修改 QA.txt 檔案
  2. Edit the QA.txt file
  3. 執行 "執行我-Execute-me-auto-update-QA-To-content.bat"
  4. Run "Execute-me-auto-update-QA-To-content.bat"
  5. 在 Chrome 擴充功能頁面重新載入
  6. Reload in Chrome extensions page
