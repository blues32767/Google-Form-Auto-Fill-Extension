<a name="繁體中文"></a>
## 🇹🇼 繁體中文說明

### 🌟 核心功能
* **智慧關鍵字對應：** 採用模糊匹配規則。例如：設定關鍵字 `mail`，即可自動對應到 `Email`、`E-mail`、`電子郵件` 等題目。
* **最長路徑優先：** 若有重複規則，程式會自動選擇最精準的項目（例如 `手機` 與 `行動電話` 同時存在時，會精準填寫）。
* **多種欄位支援：** 包含文字框、單選題、複選題（核取方塊）。
* **多組設定檔：** 可自由新增「工作」、「家裡」、「活動」等不同答案組合，隨時切換。
* **全自動操作：** 支援自動點擊「提交」、自動勾選「同意條款」或「傳送副本」。

### 📥 安裝教學
1.  前往 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/fbienifgcnebfgefaeccnmgekfcdedkc?utm_source=item-share-cb)。
2.  點擊 **「加到 Chrome」**。
3.  安裝後，點擊瀏覽器右上角的「拼圖圖示」，將本工具 **「釘選」** 到工具列。

### ⚙️ 詳細操作與規則說明
1.  **開啟面板：** 點擊工具列圖示，進入設定畫面。
2.  **設定規則：** 每一行代表一個規則，格式為 `題目關鍵字,答案`。
    * **一般文字題：** `姓名,王小明`
    * **模糊匹配：** `mail,test@example.com`（題目出現 "Email" 或 "信箱" 皆會填入）。
    * **複選題 (核取方塊)：** 若要勾選多個選項，請用逗號隔開。
        * 範例：`興趣,攝影,旅遊,美食`（自動幫你勾選這三項）。
3.  **儲存與生效：** 點擊「儲存設定」。進入 Google 表單後，確認右上角浮動工具顯示 **ON** 即可。

<a name="english"></a>
## 🇺🇸 English Guide

### 🌟 Features
* **Fuzzy Matching:** A keyword like `mail` will match `Email`, `E-mail`, or `Your Email`.
* **Checkbox Support:** Handles multiple choices with simple comma separation.
* **Profiles:** Manage multiple identity profiles for different form types.

### 📥 Installation
1.  Go to the [Chrome Web Store](https://chromewebstore.google.com/detail/fbienifgcnebfgefaeccnmgekfcdedkc?utm_source=item-share-cb).
2.  Click **"Add to Chrome"**.

### ⚙️ How to Use
1.  **Rules:** Enter `Keyword,Answer` per line.
    * Example: `Skills,Python,JavaScript` (checks both options).
2.  **Auto-Pilot:** Enable "Auto Submit" in settings to finish forms instantly.
