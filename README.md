## 🇺🇸 English Guide

### What is this?

The ultimate automation tool for Google Forms\! Stop typing your name, email, and address manually. This extension intelligently detects question keywords and fills them with your preset answers.

### Store Link 🔗

👉 [Install from Chrome Web Store](https://www.google.com/search?q=https://chromewebstore.google.com/detail/fbienifgcnebfgefaeccnmgekfcdedkc)

### Powerful Features ✨

  - 👤 **Multi-Profile Management**: Create multiple identity profiles and switch between them with one click.
  - 🚀 **Auto-Submit Mode**: Automatically clicks the "Submit" button after filling - perfect for limited slots.
  - ✅ **Auto-Checkbox**: Automatically ticks "Send me a copy" and "I agree to terms" boxes.
  - 🎯 **Smart Matching**: Uses "Longest Match First" & Case-Insensitive logic (e.g., correctly fills "Full Name" instead of just "Name").
  - ⚙️ **On-Page Widget**: A floating status bar on the form page with a quick settings button.
  - 🌐 **Multi-Language**: Supports EN, ZH, JA, KO, TH, ES, PT.

### Setup Guide 📝

1.  **Install**: Add the extension from the Chrome Web Store.
2.  **Configure**: Click the extension icon, set your language, and enter rules: `Keyword,Answer`.
3.  **Save**: Click Save.
4.  **Fill**: Open any Google Form, and the assistant will handle the rest\!
